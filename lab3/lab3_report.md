## Ход работы

### 1. Создание бакета и загрузка данных
В рамках существующего проекта был создан бакет с именем `lab3-dk`. Для хранения выбраны стандартные параметры (Standard storage class, регион us-east1). После создания в бакет были загружены графические файлы.

![Создание бакета](./screenshots/01_bucket_created.png)
![Загрузка файлов](./screenshots/02_files_uploaded.png)

### 2. Структурирование хранилища
Для организации данных внутри бакета была создана папка `memes/`. Все загруженные ранее изображения были перемещены в эту директорию.
![Структура папок](./screenshots/03_folder_structure.png)

### 3. Настройка публичного доступа
Для того чтобы файлы были доступны по прямым ссылкам без авторизации, были изменены настройки разрешений (Permissions):
*   Добавлен новый субъект: `allUsers`.
*   Назначена роль: `Storage Object Viewer`.


![Настройка доступа](./screenshots/04_public_access_check.png)

### 4. Тестирование публичных ссылок
Для проверки корректности настроек доступа были сформированы публичные URL-адреса. При переходе по ссылкам изображения успешно открываются в браузере.

![Проверка ссылки 1](./screenshots/05_meme_link_1.png)
![Проверка ссылки 2](./screenshots/06_meme_link_2.png)
